Bayes Filter (Probabilistic Robotics)

本文是阅读 《Probabilistic Robotics》 Chapter2 Recursive State Estimation 的笔记。

背景知识：

State, Control, Measurement

对于一个机器人，机器人具有与周围环境交互的能力，机器人通过它的执行部件（actuators）改变周围的环境，或者通过它的传感器（sensors）感知周围的环境。将机器人通过它的轮子改变自身在环境中所处的位置，也可以看做是改变周围环境，因为从机器人设备坐标系观察周围环境变了。

可以使用一组状态值来描述机器人在环境中的位置、姿态，环境中的任何一个物体也可以使用一组状态值来描述物体在环境中的位置、姿态。状态值不仅可以用于描述物体的位置和姿态，还可以用于描述一系列其他的内部状态，如机器人内存使用量、物体的颜色等。

机器人可以通过它的执行部件自身在环境中的位置和姿态，比如机器人发送指令给底盘，使底盘以 2m/s 的速度运动 2s，这个指令就可以看做是一组控制值。控制值造成了机器人的位置、姿态改变，所以改变了机器人的状态值。

机器人可以通过它的传感器感知周围的环境，一般来说传感器有光学相机、激光扫描仪、深度相机、声呐等。这些传感器获得的数据被称作测量值。这些测量值可以用于计算估计机器人的位置、姿态，即测量值可以计算得到状态值。

在下文中，我们使用字母 $x$ 表示机器人的状态值，一般的状态值有6个，即三维空间的3个坐标值与3个姿态值。使用字母 $u$ 表示机器人的控制值，使用字母 $z$ 表示机器人的测量值值。

Belief Distribution

Belief Distribution 是一个后验概率的概率分布。是在已经获得所有数据的条件下，状态值 $x$ 的概率分布。所有数据指的是机器人的控制值 $u$ 与测量值 $z$ 。

对于 $t$ 时刻机器人的状态值 $xt$ ，与开始时 1 时刻到 $t$ 时刻的所有控制值、测量值都有关系。从 0 时刻机器人初始位置开始，机器人通过所有的控制值 $u{1:t}$ 可以计算出机器人在 $t$ 时刻的状态值，同时所有的测量值 $z_{1:t}$ 可以验证机器人的状态值，因为机器人的控制值存在误差，如机器人运动轮打滑。

对于这个后验概率的概率分布 belief 可以写成一个条件概率：

$$ bel(x_t) = p(xt|z{1:t}, u_{1:t}) $$

如果在 $t$ 时刻不考虑测量值 $z_t$ ，只考虑控制值 $u_t$ ，在使用控制值 $u_t$ 计算出 $t$ 时刻的状态值 $x_t$ 之后再使用测量值 $z_t$ 验证得到的状态值 $x_t$ 。那么只考虑控制值 $u_t$ 得到的状态值 $x_t$ 的概率分布可以看做是一个预测 prediction ，这种方式得到的后验概率，可以写作：

$$\overline{bel}(x_t) = p(xt|z{1:t-1}, u_{1:t})$$

Markov Assumption

Markov Assumption 也可以称作是 complete state assumption。这个假设是说在知道当前的状态值的条件下，以前获得的所有数据和以后获得的所有数据之间是相互独立的。即预测未来状态值这件事情上，在当前除了当前的状态值 $x_t$ 以外，没有什么其他的数据值能够比 $x_t$ 预测得更好。也就是说 $xt$ 中蕴含了 $z{1:t}$ 和 $u{1:t}$ 的信息，当然也蕴含了 $x{t-1}$ 的信息，因为 $x{t-1}$ 是由 $z{1:t-1}, u_{1:t-1}$ 计算得到的。

贝叶斯滤波器流程

Algorithm Bayesfilter($bel(x{t-1})$, $u_t$, $z_t$):

​ for all $x_t$ do

​ $\overline{bel}(x_t) = \int p(x_t|ut, x{t-1})bel(x{t-1})\text{d}x{t-1}$

​ $bel(x_t) = \eta p(z_t|x_t)\overline{bel}(x_t)$

​ endfor

​ return $bel(x_t)$

贝叶斯滤波器数学推导

贝叶斯滤波器的目的是估计系统在各个时刻的状态值的概率分布。即已知系统在初始时刻的状态值和初始时刻到 $t$ 时刻之间各个时刻的控制命令 $u{1:t}$ 和测量值 $z{1:t}$ 的情况下，求取 $t$ 时刻系统的状态值 $x_t$ 概率分布。

这个概率分布可以用 $p(xt|z{1:t},u_{1:t})$ 表示。

而对于三个随机变量$x, y, z$，贝叶斯理论有下面的公式：

$$p(x|y,z) = \frac{p(y|x,z)p(x|z)}{p(y|z)}$$

将 $p(xt|z{1:t},u_{1:t})$ 中的 $x_t$ 对应 $x$ ，$zt$ 对应 $y$ ，$(z{1:t-1}, u_{1:t})$ 对应 $z$ ，就能写出下面的式子：

$$p(xt|z{1:t},u_{1:t})=\frac{p(z_t|xt,z{1:t-1},u_{1:t})p(xt|z{1:t-1},u_{1:t})}{p(z_t|xt,z{1:t-1},u_{1:t})} \ = \eta p(z_t|xt,z{1:t-1},u_{1:t})p(xt|z{1:t-1},u_{1:t})$$$p(z_t|xt,z{1:t-1},u_{1:t})$ 表示的是在已知 $xt,z{1:t-1},u_{1:t}$ 的条件下，$z_t$ 的条件概率。由马尔科夫假设， $xt$ 中蕴含了 $z{1:t-1},u_{1:t}$ 的信息，所以

$$p(z_t|xt,z{1:t-1},u_{1:t}) = p(z_t|x_t)$$

注意，此处的 $xt$ 指只经过 $x{t-1}$ 与 $u_t$ 计算的 $x_t$ 并没有使用 $z_t$ 。

将

$bel(x_t) = p(xt|z{1:t}, u_{1:t})$

$\overline{bel}(x_t) = p(xt|z{1:t-1}, u_{1:t})$

代入

$p(xt|z{1:t},u_{1:t}) = \eta p(z_t|xt,z{1:t-1},u_{1:t})p(xt|z{1:t-1},u_{1:t})$

可以得到：

$bel(x_t) = \eta p(z_t|x_t)\overline{bel}(x_t)$

上面这个式子指的是，在整合了 $z_t, u_t$ 之后状态值 $x_t$ 的概率分布。

对于 $\overline{bel}(x_t)$ 的计算可以用用公式

$$p(x) = \int p(x|y)p(y) \text{d}y$$

推导得到。

$$\overline{bel}(x_t) = p(xt|z{1:t-1}, u_{1:t}) \ = \int p(xt|x{t-1}, z{1:t-1}, u{1:t})p(x{t-1}|z{1:t-1}, u{1:t})\text{d}x{t-1} \ = \int p(xt|x{t-1}, z{1:t-1}, u{1:t-1})p(x{t-1}|z{1:t-1}, u{1:t-1})\text{d}x{t-1}$$

由马尔科夫假设

$$p(xt|x{t-1}, z{1:t-1}, u{1:t}) = p(xt|x{t-1},u_t)$$

所以就有

$$\overline{bel}(x_t) = \int p(xt|x{t-1},ut)p(x{t-1}|z{1:t-1}, u{1:t-1})\text{d}x_{t-1} = \int p(xt|x{t-1},ut)bel(x{t-1})\text{d}x_{t-1}$$
